# Norfolk Southern (norfolk-southern)

Norfolk Southern Corporation is one of the nation's premier transportation companies, operating approximately 19,300 route miles in 22 states and the District of Columbia. The company provides an API Resource Platform (ApiHub) offering real-time supply chain visibility.

## APIs

### Shipment Status API

Pinpoint the current location and ETA for shipments with details on commodity, origin, and destination.

### Trip Plan API

Track a shipment's progress on its planned route including current ETA, future movements, and completed movements.

### Gate Receipts API

Access gate receipt data, terminal and driver information, and pickup numbers for equipment.

## Developer Portal

Norfolk Southern's API Resource Platform is available at [developer.nscorp.com](https://developer.nscorp.com/). The portal includes self-serve tutorial videos and dedicated NS representative support.
